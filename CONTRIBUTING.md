# 參與 & 貢獻

## 我想要提 Issue

如果是漢化問題的話，請按照如下格式將問題提交至 [此處](https://github.com/screeps-cn/docs/issues) 來方便我們的檢查和修改，當然，你也可以直接在 [QQ群]((https://shang.qq.com/wpa/qunwpa?idkey=8d9a9245519f3ecc94b23fbdada6d6479d8a3330071e0d44f372bb63a372a083)) 中反映問題：

- 錯誤內容
- 我覺得應該如何翻譯（可選）
- 錯誤文章鏈接
- 官方原文鏈接

如果是官方文檔也存在該問題的話，請在 [此處](https://github.com/screeps/docs/issues) 提交 issue。

## 我想要提 Pull Request

如果你想要提交一個 PR 的話，請按照以下流程進行：

- 請確保你本地可以正常運行該項目。
- 請勿修改翻譯內容之外的文件，尤其是 `package.json` 及 `package-lock.json`，由於 npm 的原因，這兩個文件在安裝依賴後很容易被更新。
- 請在翻譯完成後通讀一遍來確定沒有錯別字等問題。
- 請解決完所有的沖突後再提交 PR。
- 提交時請正確選擇對應的 Label（PR 頁面右側），本項目一共包括下面四個可選 Label：
    - `translate`：新內容漢化
    - `fix`：錯誤勘正
    - `polish`：文本潤色
    - `sync`：同步官方文檔（由項目管理人員負責）

提交完成後我們會進行審核並合並你的 PR（一般來說很快的），並將你的信息加入到本項目的鳴謝列表中。

## 我只想翻譯內容

如果你想向本項目作出貢獻但是苦於不會 Node / 不會 git / 不會提交 pr 的話。沒有關系，你可以只翻譯內容，剩下的請聯系 QQ 群管理或者本項目管理人員，我們會幫你完成剩下的操作。

你可以在下面路徑找到文檔的存放位置，文章均使用 markdown 格式編寫：

- [介紹文檔](https://github.com/screeps-cn/docs/tree/master/source)
- [API 文檔](https://github.com/screeps-cn/docs/tree/master/api/source)

你可以根據感覺翻譯其中的內容，而不必關系這些內容是否可以正常訪問之類的問題。

同樣，一旦我們采納了你的翻譯，我們會在 commit 上注明該內容由你翻譯，並將你的信息加入到本項目的鳴謝列表中。

## 我想要參與維護本項目

歡迎常駐！你可以聯系 [團隊人員](https://github.com/orgs/screeps-cn/people) 以獲取項目的編輯限權。但是記得先多提幾個 PR 來證明你的實力哦 ~
